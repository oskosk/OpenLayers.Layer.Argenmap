OpenLayers.Layer.Argenmap

Argenmap para OpenLayers

Una clase OpenLayers.Layer.Argenmap para poder usar la infraestructura de tiles de Argenmap del Instituto Geográfico Nacional de Argentina

Ver ejemplo »

Uso básico 

map = new OpenLayers.Map( 'mapa');
layer = new OpenLayers.Layer.Argenmap();
map.addLayer(layer);
//Esta capa utiliza el SRS EPSG:3857 (equivalente al EPSG:900913) por lo tanto
//el mapa necesita reproyección de coordendas geograficas a 
//a mercator para operaciones como setCenter
map.setCenter(
    new OpenLayers.LonLat(-59, -34).transform(
        new OpenLayers.Projection("EPSG:4326"),
        map.getProjectionObject()
    ), 7
);
